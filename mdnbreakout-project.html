MDN Breakout Game

Javascript & HTML5

Via de website ww.developer.mozilla.org heb ik een tutorial gevonden waarmee je een simpele 2D MDN Breakout Game kunt maken. Deze tutorial legt stapsgewijs uit hoe je van begin tot eind deze game kunt maken met enkel Javascript code die rendered in HTML5.

Stap 1:
De eerste stap was om het canvas te maken waarop alles zou moeten komen te staan. Binnen deze stap heb ik een beetje zitten proberen om daarna de eerste stap van de tutorial klaar te krijgen. Dit was om een balletje te maken en deze weer te geven op het canvas
Stap 2:
Tijdens deze stap was het de bedoeling om middels Javascript code de aangemaakte bal in stap 1 te laten bewegen. Wat ik hier vooral geleerd heb is dat je beter verschillende functies kunt maken in plaats van 1 grote functies. Zo kun je de code makkelijker hergebruiken en is de code cleaner.
Stap 3:
Gedurende deze stap moest er gezorgd worden voor een bal die tegen de muren aan stuiterde. Dit is gedaan middels if statements. Hier heb ik al een tijdje geen gebruik van gemaakt en deed dit wel weer een beetje boven water helpen bij mij.
Stap 4:
Deze stap heeft ervoor gezorgd dat je met de pijltjestoetsen de eerder aangemaakte plank kunt bewegen. Het is namelijk geen game als er geen interactie is. Tijdens deze stap heb ik geleerd hoe boolean variabelen werken en hoe je deze moet gebruiken.
Stap 5:
Gedurende deze stap wordt er een melding gemaakt wanneer de bal de grond raakt. Dit is gedaan middels if/else statements en het gebruik van alerts.
Stap 6:
Bij deze stap wordt het stenenveld gemaakt dat je moet uitspelen met de bal. Hier is wederom gewerkt met variabelen en functies.
Stap 7:
Het stenenveld is wel aangemaakt maar de bal gaat er nog doorheen. Bij deze stap wordt ervoor gezorgd dat de stenen weg gaan wanneer de bal deze raakt.
Stap 8:
Hier wordt er een functie aangemaakt die ervoor zorgt dat je je score kunt bijhouden. Per steen die weg gaat komt er een punt bij.
Stap 9:
Bij deze stap maak ik een functie aan waarmee je ook je muis kunt gebruiken naast je pijltjestoetsen.
Stap 10:
Naast de score kun je ook meerdere levens toevoegen aan het spel. Wanneer je af gaat, zal er een leven minder worden weergegeven.

Ik heb alle code voorzien van comments en documentatie zodat een ander makkelijk kan zien wat welke functie doet. Zo is het makkelijker opnieuw te gebruiken.
