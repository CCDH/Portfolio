Semester 4 en maatwerk

Mijn maatwerk begon een beetje stroef. Ik wist niet zo goed waar ik moest beginnen. Het idee was om voor mijn development een nieuw portfolio te gaan programmeren die ik ook in semester 6 kon gaan gebruiken. Met als achterliggende gedachte dat ik me in semester 6 geen zorgen hoef te maken over het design van mijn portfolio en hier gewoon mijn werk in kon gaan stoppen.

Om dit portfolio te gaan maken wil ik gaan werken met een framework. Ik heb ervoor gekozen om gebruik te gaan maken van Bootstrap 3. Dit maakt het maken van een website die responsive moet zijn op een hoop devices een stuk makkelijker. In het begin heb ik geprobeerd de beta van Bootstrap 4 te gebruiken.

Echter zijn er een aantal dingen veranderd bij deze versie wat resulteerde in een stroeve start van mijn portfolio.

Ook zijn er nog niet zo veel voorbeelden van Bootstrap 4 en is er nog weinig documentatie te vinden. Dit kan er simpelweg voor gaan zorgen dat ik niet de juiste middelen heb om de website te maken. Ook ben ik al een beetje bekend met Bootstrap 3.

Wat is Bootstrap?
Bootstrap is een front-end framework waarmee ontwikkelaars een mooie uitgebreide (basis) set van elementen tot hun beschikking krijgen. Hiermee kan snel en eenvoudig het ‘raamwerk’ voor een responsive website gebouwd worden.

Het mooie van Bootstrap is dat het ‘Mobile First’ is opgezet. Dit betekent dat het in de basis uitgaat van een ontwikkeling op een mobiel apparaat zoals een tablet of smartphone.

Waarom Bootstrap?
- In Bootstrap zijn alle CSS-elementen al gemaakt, wat ervoor zorgt dat je zelf geen tijd verliest aan het maken van een CSS-bestand met de juise styling (mits je geen dingen custom wilt maken, dan moet je wel zelf nog een CSS erbij maken)
- Vrijwel alle elementen bestaan in Bootstrap, zoals tabellen, knoppen, formulieren, labels etc.
- Bootstrap wordt op vrijwel elke browser ondersteund
- Bootstrap is geoptimaliseerd voor zowel desktop, als tablets en mobiele telefoons
- Duidelijke documentatie waardoor je, ook wanneer je minder kennis van Bootstrap hebt, snel aan de slag kunt om je website te bouwen.

Omdat ik in semester 4 te weinig code heb laten zien moet er in dit portfolio dus meer code komen te staan. Het gaat hier voornamelijk om Javascript. Natuurlijk mag er ook op een andere manier aangetoond worden dat ik kennis over development beschik.

Wat is Javascript?
JavScript is een scripttaal waarvan de code wordt uitgevoerd in de webbrowser van de bezoeker van een website. De technologie wordt veel gebruikt voor het creëren van een interactieve interface en dynamische gebruikerservaring binnen webpagina’s. Behalve op websites wordt JavaScript ook gebruikt in applicaties en in de aansturing van verschillende elektronica.

Wat kun je met Javascript?
In zijn algemeenheid kun je zeggen dat je met JavaScript je site interactief kunt maken. Zo kun je de bezoekers een vraag stellen of browsergegevens achterhalen waarna deze gegevens gebruikt worden op je site.

Bijvoorbeeld: als een bezoeker je site bezoekt, ziet hij een venster waarin hem naar zijn naam gevraagd wordt. Nadat hij de naam heeft ingetypt, wordt op de site, op alle gewenste plaatsen, de bezoeker met zijn (voor)naam aangesproken.

Nu is dit misschien niet zo'n nuttige toepassing van JavaScript (misschien zelfs wel irritant), maar het laat wel goed zien dat JavaScript je site interactief maakt.

Om meer kennis van code te krijgen heb ik middels lynda.com een cursus gevolgd voor Angular. Van de theorie heb ik dingen genoteerd en heb ik een hoop aantekeningen gemaakt. Naarmate de cursus vorderde kwam ik erachter dat het allemaal steeds lastiger werd en ik zelf niet voldoende kennis had om hiermee door te gaan. Ik moest een stapje terug doen. Ik moest eerst meer kennis verschaffen over de basis van Javascript. Ik ben via lynda.com de cursus Javascript begonnen. De theorie heb ik gevolgd en ik heb aantekeningen gemaakt.

Gedurende deze cursus ben ik begonnen met het maken van een kleine library. Dit had ik niet gedaan tijdens semester 4. Het gaat hierbij om een library waarmee je een SVG-object kunt laten draaien, de transparantheid en de grootte kunt aanpassen middels Javascript code. De code in de Javascript is voorzien van comments zodat deze makkelijk te hergebruiken is. Je hoeft hier alleen maar de selector aan te passen naar iets wat je zelf wilt en de rest wijst zijn eigen. Deze library is terug te vinden op de homepage van mijn portfolio.
