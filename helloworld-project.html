Javascript

Om nog meer te oefenen met JavaScript heb ik een kleine testpagina gemaakt met verschillende functionaliteiten die mogelijk zijn met Javascript. Deze pagina bevat minimale HTML omdat vrijwel alles via de JavaScript file is gedaan.

Ik heb hier gewerkt met querySelectors waarmee ik verschillende HTML-elementen kan aanroepen met behulp van JavaScript. Hier heb ik nog nooit eerder mee gewerkt maar ik snap nu wel de waarde ervan en dat het voor sommige dingen erg handig kan zijn om deze te gebruiken.

Verder heb ik gewerkt met de on-clickfunctie binnen JavaScript. Dit heb ik al eerder gedaan alleen zorgt de on-clickfunctie die ik nu gebruikt heb ervoor dat wanneer je op een element klikt deze de afbeelding veranderd naar een nieuwe afbeelding.

Ook heb ik kennis gemaakt met de localStorage. Waarmee je als gebruiker je naam kunt invullen wanneer je op de pagina komt zodat deze een berichtje met naam weergeeft op het scherm. Je kunt daarna ook de naam aanpassen door op de knop onderaan de pagina te drukken. Deze knop zorgt ervoor dat er een pop-up venster komt waarmee je de naam kunt wijzigen.

Het enige probleem is dat je wanneer je de pagina vernieuwd de button niet meer werkt en je ook je naam niet meer kunt invullen. Om dit op te lossen moet je je geschiedenis verwijderen en opnieuw de pagina openen.
